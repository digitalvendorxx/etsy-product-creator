## Notes on Data Reliability

- All DTF shop revenue figures are UNVERIFIED — Etsy 403-blocks all automated fetches; eRank/eHunt require authenticated login.
- Consumable cost ranges are consistent across 3+ sources (S10, S14, S17, S18), giving moderate confidence in the $1.00-$2.00 per A3 print range.
- Equipment prices from vendor sites (S5, S7) represent current retail; third-party reviews (S4, S6) tend to match within 10-15%.
- "Plantify Pro" as described in the README does not exist. The correct tool is "Planify Pro" — a planner design tool with no DTF function. Separately, Printify Pro = Printify's premium plan ($29/mo). Neither is a DTF-specific tool.
