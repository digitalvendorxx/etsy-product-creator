# 02 — Listing: Operating Rules

## 1. TL;DR

- Title: lead with the strongest keyword phrase in the first 3-5 words; stay under 14 words (140 chars max); no subjective adjectives; no repetition of the same word in more than 2x.
- Tags: always use eHunt first; 13 tags max, each 20 chars max; long-tail, low competition, high score; no single lead word dominating more than 40% of the tag set.
- Description: emotional hook -> product specs -> sizing -> ordering -> shipping; front-load keywords in first 160 chars.
- Photos: up to 10; first photo = clean product shot; lifestyle, scale, detail, variant photos follow; min 2000px on shortest side.
- Attributes: fill ALL applicable — they act as extra tags and affect relevance matching.
- Category: choose the most exact match possible — more specific = better relevance signal.
- Processing time: keep as short as realistically achievable; late shipments hurt ranking.
- New listing boost: a few hours to a few days only. Organic ranking comes from engagement, not freshness.
- Renew when a listing is performing well and approaches expiry. Relist (new listing) only when the listing has accumulated no quality score worth preserving.
- Refresh (edit in place) when you want to improve a listing without losing its quality score. Make changes in small batches; allow 2-4 weeks for algorithm to re-evaluate.

For fee structure, search ranking factors overview, and Star Seller criteria, see [../01-how-etsy-works/rules.md](../01-how-etsy-works/rules.md).

## 2. Listing Creation Checklist

Run through in order before publishing any new listing.

1. **Research tags first (eHunt)** — generate a broad keyword from the design (e.g. "funny mom shirt"), run eHunt, capture the top-scoring, low-competition, long-tail results. All 13 tags come from here.
2. **Write the title** — use the strongest keyword phrase from step 1 as the opener; keep to 14 words or fewer; no subjective words; no repeated lead words more than 2x. See Title Formula below.
3. **Check Etsy title tips** — when the in-editor Tips panel appears, apply all bullets (length, repetition, subjective words). Never auto-apply the "Suggested" title — always verify it matches the actual design.
4. **Write the description** — follow the Description Template (section 5). Front-load the main keyword within the first 160 characters.
5. **Enter all 13 tags** — formatted as comma-separated single line; each tag max 20 characters. Pull from eHunt results only; no invented tags.
6. **Upload 10 photos** — in the order specified in the Photo Checklist (section 6). First photo is the hero/product shot. All images min 2000px on shortest side.
7. **Add a listing video if available** — 5-15 seconds, no audio, 1080p, max 100 MB. Square or vertical (4:5) preferred for mobile. Text overlays with key specs recommended since there is no audio.
8. **Select the most specific category** — drill down to the leaf-level category, not a parent/general one.
9. **Fill ALL attributes** — size, color, material, occasion, style, holiday, etc. Each attribute is an extra relevance signal.
10. **Set variants** — if the item comes in multiple sizes or colors, configure as variants (not separate listings). Etsy tracks inventory and conversions per variant.
11. **Add personalization field** — if the item requires buyer input (name, date, custom text): set as Required so the order cannot begin without it. If it is optional (gift note, no-text version available), set as Optional. Keep the prompt clear with one request and an example format.
12. **Set processing time** — set the shortest time you can reliably meet. Processing time + carrier transit = Estimated Delivery Date shown at checkout. Late shipments against your stated time hurt shop quality score and ranking.
13. **Set shipping** — offer free shipping to US buyers where margin allows; free shipping on orders $35+ via the Free Shipping Guarantee improves US search placement. See ../01-how-etsy-works/rules.md for full shipping rules.
14. **Set return policy** — even "no returns accepted" is better than blank. A stated return policy improves search placement.
15. **Assign to a shop section** — choose the broadest fitting section (see Section Rule below). Only create a new section if nothing fits; use a name broad enough to hold many future designs.
16. **Price check** — POD t-shirts: minimum $24-28 to maintain ~40% margin after Etsy fees and print cost. Use charm pricing (.99 or .95 ending) for perceived value. Consider anchor pricing if offering both tee + sweatshirt (show the higher-priced item alongside the lower one).
17. **Final review** — read the listing as a buyer. Does the first 160 chars of description include the main keyword? Is there any title tip still unfixed? Are all 13 tags under 20 chars?

## 3. Title Formula

### Template

```
[Primary keyword phrase] | [Key attribute 1] [Key attribute 2] — [Secondary keyword or audience]
```

- Lead with the strongest buyer search phrase (verb + noun or adjective + noun + product type).
- Use a pipe ( | ) or dash ( - ) as separator between the primary phrase and secondary details.
- Total: 14 words or fewer (Etsy hard recommendation, verified 2026-04-27).
- No subjective adjectives ("cute", "amazing", "beautiful", "perfect") — move them to description.
- No sales or shipping info in title.
- No repeated lead word more than 2x in the full title. If Etsy flags a word under the Tips panel, it must appear only once in the revised title.
- The most important keyword must appear in the first 3-5 words; only the first ~50-60 characters show before search truncation.

### Examples (T-shirt POD context)

1. `Funny Mom Word Cloud Shirt | Gift for Mom Birthday`
2. `Royal Blue Queen Birthday Tee - Crown Design Women Gift`
3. `Personalized Dad T-Shirt Custom Kids Names - Father's Day`

### What NOT to do

- `Dad T-Shirt Father Shirt Personalised, Father's Day Gift Personalised Dad T-Shirt, Dad Statement Shirt...` (repetition, too long, keyword stuffing)
- `Amazing Cute Mom Gift Tee Perfect For Mothers Day Beautiful Design` (all subjective adjectives, no product clarity)

## 4. Tags Strategy

*Source rule from memory: eHunt research is mandatory before finalizing tags. See also: feedback_ehunt_tags_titles.md (integrated below).*

### Selection method

1. Derive 2-3 general keywords from the title (e.g. "mom shirt", "birthday tee", "funny gift").
2. Run each in eHunt. Capture results.
3. Pick the best 13 tags that meet ALL of these criteria:
   - Competition: LOW
   - Score: HIGH
   - Must be LONG-TAIL (2+ words preferred; avoids fighting broad single-word terms)
   - Weekly sales: HIGHEST available
   - Favorites and views: HIGH
   - Relevant to the actual design — no tangential tags just for traffic
   - Each tag max 20 characters (count every character including spaces)
4. Check lead-word diversity: no single root word (e.g. "mom", "royal", "birthday") in more than 5-6 of the 13 tags (~40%). Swap excess for scored synonyms from the same eHunt pool (e.g. "queen", "crown", "gold" instead of more "royal").
5. Output format: comma-separated single line, no bullets, no numbering, no line breaks.

### Format example
```
funny mom tee, mama gift, mom birthday, gift for her, word cloud shirt, mother gift, mom life tee, sarcastic mom, cool mom shirt, mom shirt gift, graphic tee mom, mom humor shirt, mommy gift idea
```

### Hard limits (verified 2026-04-27)
- MAX 13 tags per listing
- MAX 20 characters per tag (count carefully; "matching family shirts" = 22 chars — invalid)
- Never invent tags not grounded in eHunt research

## 5. Description Template

*Source rule from memory: feedback_etsy_description_model.md (integrated below).*

All descriptions follow this exact structure. Customize only sections marked [CUSTOMIZE].

### SEO requirement (Etsy Seller Handbook)
- Front-load the main keyword naturally within the first 160 characters of the description.
- Natural language, no keyword stuffing.
- Describe the actual design visually — not just generic product copy.

### Structure

**[CUSTOMIZE] Paragraph 1 — Emotional hook**
2-3 sentences. Conversational. Speaks to the target customer's identity or lifestyle. Weaves in the main keyword naturally in the first sentence. Describes what the design looks like and what mood it evokes.

**[CUSTOMIZE] Paragraph 2 — Style and quality**
Mention available styles (tee + sweatshirt), briefly describe feel/quality, end with a durability note.

**[CUSTOMIZE] Gift occasion line**
One sentence listing relevant gift occasions (birthday, Mother's Day, Christmas, etc.).

## 6. Photo Checklist

All images must be min 2000px on the shortest side, 72 PPI minimum, high resolution.

| Slot | Type | Purpose |
|---|---|---|
| 1 (hero) | Clean product shot (front, on neutral or branded background) | First impression; only this one shows in search results |
| 2 | Lifestyle photo (product worn/in use in context) | Click-through rate — lifestyle photos achieve up to 5x higher CTR than plain mockups (verified signal per multiple sources) |
| 3 | Back design (if front+back listing) | Show both sides — critical for front/back listings to avoid confusion |
| 4 | Size/scale reference | Flat lay with a ruler or size chart overlay |
| 5 | Detail close-up | Show print quality, texture, design detail |
| 6-8 | Color variants | Show different color options available (one mockup per main color group) |
| 9 | Sweatshirt version (if offered) | Buyers often compare; having both garments builds trust |
| 10 | Gift packaging or lifestyle context photo | Reinforces gifting use case |

**Rules:**
- Never use a collage as the first photo — it is confusing on small screens and in search.
- First photo must show the actual product the buyer will receive (not a rendering-only image for POD — stylized mockups are acceptable if they realistically represent the final product).
- Add a listing video if you have one (5-15 sec, 1080p, no audio, max 100 MB). Square (1:1) or vertical (4:5) aspect ratio preferred.

## 7. Tracking Dashboard

Check stats weekly; do a monthly deep-dive for strategic decisions. Always analyze minimum 30-day windows; compare year-over-year when seasonality applies.

### Metrics to watch

| Metric | Where in Etsy Stats | What it means |
|---|---|---|
| Impressions | Listing-level stats | How often your listing appeared in search/browse results |
| Visits (clicks to listing) | Listing-level stats | How many people clicked through to view the listing |
| Click-through rate (CTR) | Impressions / Visits | Low CTR = thumbnail or price problem, not a keyword problem |
| Conversion rate | Orders / Visits x 100 | The single most important metric |
| Favorites | Listing-level stats | Engagement signal; also feeds Etsy ranking algorithm |
| Revenue per listing | Orders x price | Identifies your actual revenue drivers |
| Traffic source | Shop Stats > Traffic | Shows how buyers are finding you (Etsy search, direct, social, Etsy Ads) |

### Conversion rate benchmarks (POD shops)

| Rate | Status | Action |
|---|---|---|
| Below 1% | Critical | Immediate intervention — review price, photos, title, description |
| 1-3% | Average for POD | Optimize incrementally; test thumbnail or price changes |
| Above 3% | Strong | Scale — create design variations, replicate the formula |

### Detecting listing decay

A listing is decaying when impressions fall over a sustained 14-30 day period with no seasonal explanation. Check for these patterns:

| Pattern | Diagnosis |
|---|---|
| Low impressions + low visits | Keyword/SEO problem — title and tags not matching buyer searches |
| High impressions + low CTR | Image or price problem — thumbnail not compelling; price too high vs. competitors |
| High visits + low orders | Trust or friction problem — unclear description, no reviews, confusing variants |
| Declining impressions over 30 days | Listing quality score dropped; engagement signals weakened |

## 8. Refresh Decision Tree

```
Is the listing still active (not expired)?
  YES →
    Has it made sales and accumulated engagement (clicks, favorites, purchases)?
      YES →
        Is it performing well now (conversion rate in target range, stable impressions)?
          YES → Leave it alone. Don't fix what isn't broken.
          NO  → REFRESH IN PLACE: edit title, tags, or photos in small batches.
                Change 2-3 elements max per update. Wait 2-4 weeks to measure impact.
                Start with worst-performing listings, not your best performers.
      NO (no meaningful sales, no engagement) →
        Has the listing expired or is it about to expire?
          YES → RELIST as new: no quality score to preserve; a fresh listing loses nothing.
          NO  → REFRESH IN PLACE first. If no improvement after 30 days, RELIST.
  NO (expired) →
    Did it have strong historical engagement?
      YES → RENEW ($0.20 fee): preserves the accumulated quality score.
      NO  → RELIST as new.
```

### Renewal cost vs. SEO benefit

- Cost: $0.20 per renewal.
- Benefit of renewing a performing listing: preserves internal quality score (clicks, favorites, purchase history) that a new listing would need to rebuild from zero.
- Benefit of renewing a dead listing: only a small temporary freshness boost (hours to a few days); not worth repeated renewals as a strategy.
- Verdict: Renew good listings when they expire. Do NOT renew dead listings repeatedly as a ranking hack — Etsy explicitly states this is not an effective long-term strategy and can accumulate fees without benefit.

### Freshness boost window (verified 2026-04-27)

- New listing boost: a few hours to a few days (Etsy's stated purpose is to gather initial engagement data, not to sustain visibility).
- Renewed listing boost: smaller than new listing; similar duration (hours to a few days).
- Do not time listings for launch-day ranking — organic engagement drives sustained visibility, not freshness.

### Safe refresh rules

- Change a maximum of 2-3 elements per update (e.g. thumbnail + title, or 3 tags).
- Never bulk-update the entire shop at once — you cannot isolate what worked.
- Never change a high-performing listing just because its eHunt/Marmalead grade is low. Something is working. Only change if metrics are declining.
- Meaningful changes that trigger a new algorithm evaluation: rewritten title, new thumbnail, updated attributes, repriced listing.
- Micro-edits (shuffling tag order, minor description tweaks) do NOT trigger re-evaluation.
- Wait 2-4 weeks after any change before evaluating impact.

## 9. Common Mistakes (What NOT to Do)

1. **Keyword stuffing in title** — "Mug Coffee Mug Cup Funny Cat Mug Gift For Her Cat Lover Mug" is penalized; write naturally.
2. **Repeating the same lead word in >40% of tags** — diversify synonyms; Etsy's own warning flags this in-editor.
3. **Auto-applying Etsy's "Suggested" title** — it often hallucinates features (personalization, birth flowers) the design does not have. Always verify before applying.
4. **Ignoring Etsy's "Tips" bullets** — these are deterministic SEO rules, not suggestions. Apply all of them every time.
5. **Leaving attributes blank** — attributes are additional relevance signals identical to tags. Blank attributes waste free ranking weight.
6. **Choosing a parent/general category** — always drill to the most specific leaf category. "Clothing" vs. "Women > Tops & Tees > T-Shirts" is not equivalent.
7. **Using 1-2 word broad tags** — "gift", "shirt", "tee" compete against millions of listings. Use long-tail phrases.
8. **Exceeding 20 chars on a tag** — Etsy silently truncates or rejects; count every character before entering.
9. **Relisting a good listing instead of renewing** — you lose the accumulated quality score. Always renew performing listings when they expire.
10. **Renewing dead listings repeatedly as a ranking strategy** — provides only hours of minor boost; costs $0.20 each time; does not fix underlying SEO problems.
11. **Using a collage as the first photo** — confusing on mobile screens; harms CTR in search.
12. **Over-optimizing a successful listing** — if conversion rate is above 3% and impressions are stable, don't change anything just because an SEO tool gives it a low grade.
13. **Setting processing time too long** — longer processing time = later EDD shown at checkout = fewer conversions. Set the shortest time you can reliably meet.
14. **Creating a too-narrow shop section** — "Sassy & Dark Humor Tees" wastes a section slot; "Funny & Attitude Tees" covers more products with the same slot.
15. **Titles longer than 14 words** — Etsy explicitly recommends 14 words or fewer; longer titles do not rank better and hurt readability.

## 10. Shop Section Rule

*Source rule from memory: feedback_etsy_sections.md (integrated).*

Etsy limits the number of sections a shop can have. When assigning a listing to a section or creating a new one:
- First, try to fit the design into an existing section using the broadest applicable name.
- Only create a new section if nothing fits.
- New section names must be broad enough to hold many future designs (e.g. "Funny & Attitude Tees", not "Sassy Dark Humor Tees").
- Never create a section for a single niche design — it wastes a permanent slot.
